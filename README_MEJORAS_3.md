# ContaCurso - Mejoras 3

Incluye:

- Vista de morosidad desde el menú principal.
- Estado de cuotas con columna de colegio para contexto multi-colegio.
- Exportación Excel de estado de cuotas/morosidad.
- Comprobante PDF por pago con folio ContaCurso.
- Filtros de cuotas compatibles con admin global, colegio y curso.

Notas:

- No borra la base actual.
- Mantiene compatibilidad con alumnos y pagos existentes.
- Para ver toda la información como admin, usa el filtro "Todos los colegios".
