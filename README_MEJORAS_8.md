# ContaCurso - MEJORAS 8

Cambios incluidos:

- Panel de plataforma para administrador global (`/admin/plataforma`).
- Plan por colegio: `basico` o `pro`.
- Estado de suscripción por colegio: `activo` o `vencido`.
- Fecha de vencimiento de suscripción.
- Modo solo lectura para colegios vencidos: usuarios del colegio no pueden registrar cambios si la suscripción está vencida.
- Métricas globales por colegio: alumnos, usuarios, pagos y total registrado.
- Migración compatible: se agregan columnas nuevas sin borrar datos existentes.

Nota: el usuario global sigue siendo `admin`.
